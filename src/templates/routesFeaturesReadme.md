# (features) routes

The directory `src/routes/(features)` is **generated** by `scripts/generate-features.js`.

- Do **not** edit files in `src/routes/(features)` manually.
- Any manual changes will be overwritten the next time the generator runs.

If you need to change what gets generated, update the generator (`scripts/generate-features.js`) and/or the templates it uses (see `src/templates`).
